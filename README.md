# SpringBoot + Vue3 在线视频论坛平台项目
>前言：此项目为本人与同学为完成课设任务在6个月时间里“赶制”出来的一个小项目，所以代码中难免会出现纰漏或者bug，请大家体谅。

## 项目名称：MiliMili大学生在线学习教育平台
- 使用Java语言，打造了一款混合式在线论坛，内容包括视频，文件和论坛本体。
- 使用前后端分离的开发方式，分为客户端和管理端。

## 技术栈
- 前端（web端）：
  - Vue3全家桶 + Element-Plus + Axios等
- 后端（服务端）：
  - SpringBoot3 + MyBatis + JWT + IKAnalyzer等
- 数据库：
  - MySQL + lucene
- 开发环境：
  - NodeJs: 22.12.0 + JDK: 1.8

## 项目功能
- 首页内容个性化推荐
- 用户注册登录
- 用户权限鉴别
- 个人中心信息修改
- 视频投稿
- 帖子提交
- 文件上传
- 视频帖子文件审核
- 内容搜索
- 评论系统（含多级评论）
- 点赞收藏系统
- 个人课程表系统
